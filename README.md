# 🎯 Unear - 위치 기반 멤버십 혜택 서비스

## 📋 서비스 소개

**Unear**는 사용자의 위치 정보를 활용하여 주변 매장의 할인 혜택과 쿠폰을 제공하는 통합 멤버십 플랫폼입니다.

### 🌟 주요 특징
- **위치 기반 서비스**: 현재 위치 주변의 매장 정보와 혜택 제공
- **통합 멤버십**: 다양한 브랜드의 멤버십을 하나의 앱으로 관리
- **쿠폰**: 매장별 쿠폰 제공
- **개인화된 추천**: 사용자 패턴 분석을 통한 맞춤형 혜택 추천
- **소셜 로그인**: Google, Kakao, Naver 간편 로그인 지원

## 🏗️ 기술 스택

- **Framework**: React 19.1.0 + TypeScript
- **Build Tool**: Vite 7.0.0
- **Styling**: Tailwind CSS 3.4.17
- **State Management**: Zustand 5.0.6
- **Routing**: React Router DOM 7.6.3
- **HTTP Client**: Axios 1.10.0 + TanStack React Query 5.83.0
- **Testing & Docs**: Storybook 9.0.17

## 🚀 서버 실행 방법

## 🚀 빠르게 실행하기 (Docker 사용)

### ✅ 1. 레포지토리 클론

```bash
git clone https://github.com/your-org/unear-frontend.git
cd unear-frontend
```

### ✅ 2. Docker 이미지 빌드

```bash
docker build -t docker/unear-fe -f Dockerfile .
```

### ✅ 3. Docker 이미지 실행

```bash
docker run -p 3000:4000 docker/unear-fe
```

➡ 브라우저에서 접속: [http://localhost:3000](http://localhost:3000)

## 📱 페이지별 기능 소개

### 🏠 메인 페이지
- **멤버십 등급 표시**: 사용자의 현재 멤버십 등급 (우수/VIP/VVIP)
- **바코드 정보**: 사용자 바코드 조회 
- **이번주니어 배너**: 이번주니어(이벤트) 소개 및 연동 배너
- **스토리 배너**: 스토리 페이지 소개 및 연동 배너
- **혜택 안내 배너**: 혜택 페이지 소개 및 연동 배너

### 🎫 혜택 안내 페이지
- **브랜드별 혜택 목록**: 카테고리별 브랜드 분류 (카페, 음식, 쇼핑 등)
- **할인율 정보**: 각 브랜드별 멤버십 등급에 따른 할인율
- **혜택 상세 보기**: 브랜드별 상세 혜택 정보
- **카테고리 필터**: 음식, 카페, 쇼핑, 뷰티 등 카테고리별 필터링

### 🗺️ 지도 페이지 
- **카카오맵 연동**: 실시간 지도 서비스
- **현재 위치 표시**: GPS 기반 사용자 위치 표시
- **주변 매장 검색**: 반경 내 혜택 가능 매장 표시
- **매장 상세 정보**: 운영시간, 전화번호, 주소, 할인 정보, 쿠폰
- **즐겨찾기 기능**: 자주 가는 매장 북마크
- **필터링**: 카테고리별, 거리별 매장 필터
- **쿠폰 정보**: 다운로드 받은 내 쿠폰 확인, 가까운 매장 정보 및 쿠폰 확인(다운로드 연동)
- **이벤트 정보**: 이벤트 지역 안내 및 지도 마커 설명 
- **바코드 정보**: 사용자 바코드 조회 

### 📖 스토리 페이지 
**개인화된 소비 분석 및 추천 시스템**
- **소비 진단**: 개인 소비 성향 분석
- **메인 스토리** : 결제 내역 연동 및, ai 활용 분석내용 제공 및 관련 스토리 이미지 연출
- **히스토리 기능** : 과거 3개월분의 스토리 내용 확인 가능
- **추천 서비스** : AI 기반 맞춤 매장 추천

### 🎮 주니어 페이지
**게임화된 멤버십 경험**
- **이벤트 배너**: 주니어 전용 이벤트 안내
- **스탬프**: 매장 방문 시 스탬프 획득
- **룰렛**: 룰렛 돌리기 시스템
- **이번주니어 전용 지도**: 이벤트 지역 안내 지도
- **오늘의 선착순 쿠폰**: 매일 제공되는 선착순 특별 혜택
- **이번 주니어 매장 안내 리스트**: 이번주니어 이벤트에서 진행중인 매장 안내

### 👤 마이 페이지
**개인 정보 및 사용 현황 관리**

#### 마이 페이지 메인
- **사용자 프로필**: 이름, 등급
- **월별 할인 차트**: 월간 할인 금액, 최근 5개월간 할인 내용 막대 그래프 자료 제공
- **최근 사용 내역**: 최근 3개 결제 기록
- **계정 관리**: 비밀번호 변경, 로그아웃

#### 통계 상세
- **월별 할인 차트**: 최근 5개월 할인 금액 그래프
- **카테고리별 분석**: 음식, 카페, 쇼핑 등 카테고리별 사용 현황
- **할인 금액 누적**: 총 절약 금액 및 사용 횟수
- **상세 분석**: 전월 동월 대비

#### 사용 내역 
- **전체 사용 내역**: 시간순 정렬된 모든 결제 기록
- **필터링**: 기간별, 카테고리별
- **사용 통계**: 선택한 기간의 결제 기록 확인

#### 즐겨찾기
- **즐겨찾기 매장 목록**: 북마크한 매장들의 리스트
- **매장 정보**: 운영시간, 거리, 전화번호
- **무한 스크롤**: 페이지네이션 없는 부드러운 스크롤

#### 내 쿠폰
- **사용 가능 쿠폰**: 다운로드한 쿠폰들의 목록
- **만료 임박 알림**: 만료 예정 쿠폰 우선 표시
- **바코드 표시**: 쿠폰 사용을 위한 바코드 확인 가능 모달
- **쿠폰 상세**: 사용 조건, 유효기간, 할인 내용

### 🔐 인증 페이지들

#### 로그인
- **일반 로그인**: 이메일/비밀번호 로그인
- **소셜 로그인**: Google, Kakao, Naver 간편 로그인
- **로그인 유지**: 리프레시 토큰을 통한 자동 인증

#### 회원가입 
- **이메일 인증**: 이메일 인증코드 발송 및 확인
- **개인정보 입력**: 이름, 성별, 생년월일, 전화번호
- **비밀번호 설정**: 보안 규칙에 따른 비밀번호 생성

#### 추가 정보 입력 
- **OAuth 사용자 전용**: 소셜 로그인의 경우 추가 정보 입력

#### 비밀번호 변경 
- **현재 비밀번호 확인**
- **새 비밀번호 설정**

## 🔧 주요 기능 및 특징

- **OAuth2 연동**: Google, Kakao, Naver 소셜 로그인

- **실시간 위치**: GPS를 통한 정확한 현재 위치 파악
- **주변 매장 검색**: 반경 내 혜택 가능 매장 실시간 검색
- **거리 계산**: 사용자 위치에서 매장까지의 정확한 거리 표시
- **지도 인터랙션**: 확대/축소, 드래그, 매장 마커 클릭

- **맞춤 추천**: 사용자 선호도 기반 매장 추천
- **스토리텔링**: 데이터를 스토리로 풀어낸 개인화된 리포트

### 🎮 게임화 요소
